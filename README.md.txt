# CS2 Calculator
This calculator aims to : 
Addition of two numbers
○ Subtraction of two numbers
○ Multiplication of two numbers
○ Division of two numbers
○ Factorial calculation
○ Greatest Common Divisor (GCD) calculation
○ Least Common Multiple (LCM) calculation

1. To create a build directory, run mkdir build.
2. Use cd build to navigate to the build directory.
3. To create the Makefile, run cmake...
4. To compile the project, run make.
5. To run the executable, type ./calculator.
